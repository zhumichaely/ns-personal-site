About This Website

I built this website in order to showcase some things I learned how to do with web design, as well as serve as a place to provide some personal insight on other projects I've done.

As far as the showcasing part, the site is relatively basic but can hopefully demonstrate that I'm able to make something.

The page itself doesn't use any client-side framework, just JavaScript, so the animations and other non-basic behavior you might observe were all written by yours truly (do not append "/rainbow.js" to the URL: you won't like what you see). However, I do use the Express NPM package for a "backend" as I wanted to make this site non-static. It shouldn't be too hard to find out how so--that said, if you're reading this, you haven't.

At this point I'm fairly confident that you aren't on a mobile device. If you are, you own an impressively large tablet. Props. I tried to be as mobile-friendly as possible; try viewing this site on a smartphone to see the changes I made to do so!

As for the rest of the site, there's nothing much to say. It's just a basic website. I was able to make the "frontend" relatively easily, and you can find its static version hosted on GitHub Pages.
