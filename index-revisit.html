About This Website

I built this website in order to demonstrate some competence when it comes to web design, as well as provide some personal insight on other projects I've done.

As far as the demonstrating part, the site is relatively basic but should serve as sufficient proof that I'm able to make something.

The page itself doesn't use any client-side framework--just vanilla JavaScript--so the animations and other non-skeletal behavior you might observe were all written by yours truly, manually. However, I do use the Express NPM package on the server side. I wanted to distinguish this site as non-static (competence!), and I can think of many things I'd rather do than handle the boilerplate involved with raw HTTP requests. It shouldn't be too hard to find out how I treat different users differently, and if you're reading this, there's a chance you already have (hint: scroll up).

This site is secure. Though there is no opportunity for you to somehow divulge your credit card information here, the peace of mind is nice, isn't it? No one at the airport can eavesdrop on what you're doing here--you can browse shame-free! I mention this not because having Transport Layer Security is particularly impressive, but because there is a particularly impressive amount of sites that don't. A scary amount of the servers run by my own school have a deprecated implementation or no encryption at all. A basic, modern certificate is free in cost and nearly free in time to acquire!

At this point I'm fairly confident that you aren't on a mobile device. If you are, you own an impressively large tablet. Props. I tried to be as mobile-friendly as possible; try viewing this site on a smartphone to see how I did!
