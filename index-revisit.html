About This Website

I built this website in order to showcase some things I learned how to do with web design, as well as serve as a place to provide some personal insight on other projects I've done.

As far as the showcasing part, the site is relatively basic but can hopefully demonstrate that I'm able to make something. The page itself doesn't use any client-side framework, just JavaScript, so the animations and other non-basic behavior you might observe were all written by yours truly (do not append "/rainbow.js" to the URL: you won't like what you see). However, I do use the Express NPM package as I wanted to make this site non-static. It shouldn't be too hard to find out how the page changes. In fact, if you're reading this, there's a chance you have already (hint: scroll up).

If you've come here for my take on my other projects, simply keep reading!
